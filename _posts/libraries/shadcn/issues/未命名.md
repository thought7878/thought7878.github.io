# shadcn-ui 适合新手开发者的 Issue 推荐列表

## 重要说明

经过全面搜索，我发现 shadcn-ui 仓库**没有使用 "good first issue" 或 "beginner friendly" 标签**。因此，我根据以下标准为你筛选了适合新手的 issue：

- ✅ 状态：Open（开放）且**未被 assignee**（无人认领）
- ✅ 评论数少（0-2条），说明还没有人在处理
- ✅ 问题范围明确，不需要深入理解核心架构
- ✅ 技术栈要求：React + TypeScript + Tailwind CSS 基础

## 🔧 提交 PR 前的准备步骤

1. **Fork 仓库**
   ```bash
   gh repo fork shadcn-ui/ui --clone
   cd ui
   ```

2. **安装依赖**
   ```bash
   pnpm install
   ```

3. **创建分支**
   ```bash
   git checkout -b fix/issue-xxxx
   ```

4. **提交前先测试**
   ```bash
   pnpm validate:registries  # 注册表相关修复
   ```

5. **提交 PR**
   - 在 issue 中评论 "I'd like to work on this" 表明你在处理
   - 提交时引用 issue 编号（如 `Fixes #10955`）
