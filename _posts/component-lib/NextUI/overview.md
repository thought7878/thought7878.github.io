# 目录结构

- apps

- packages
  
  - components：avatar/button等组件的独立package
  
  - core
    
    - react：package，导入components下的所有组件的package，再统一导出，只有这一个作用
    
    - system
    
    - system-rsc
    
    - theme：主题。NextUI 主题原语，在这里可以找到 TailwindCSS `plugin`、`colors`、`layout`、`spacing`、`typography` 和一些其他实用程序。
  
  - hooks
  
  - storybook
  
  - utilities
