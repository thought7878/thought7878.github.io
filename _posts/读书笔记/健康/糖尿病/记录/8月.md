| 状态       | 正常范围           | 糖尿病目标范围        |
| -------- | -------------- | -------------- |
| 空腹血糖     | 3.9～6.1 mmol/L | 4.4～7.0 mmol/L |
| 餐后2小时血糖  | <7.8 mmol/L    | <10.0 mmol/L   |
| 夜间/空腹最低值 | ≥3.9 mmol/L    | ≥4.4 mmol/L    |
| 餐前血糖     | 4.4～7.0 mmol/L | 4.4～8.0 mmol/L |
📌 如果你正在使用胰岛素或降糖药，务必遵循医生设定的目标范围。
