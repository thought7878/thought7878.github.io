### 布局相关面试题 
[02:46](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=3264999c3fac4b7f92cd74b6c94752fe#?seek_t=166)

- **盒模型宽度如何计算**  
    盒模型由内容（content）、内边距（padding）、边框（border）、外边距（margin）组成，需明确总宽度的计算方式。
- **margin纵向重叠问题**  
    多个纵向排列元素之间的margin会发生合并现象，需理解其机制。
- **margin负值处理**  
    margin为负值时会影响元素的位置偏移，如`margin-left: -100px`会向左移动100像素。
- **BFC的理解与应用**  
    BFC（Block Formatting Context）是一个独立渲染区域，影响子元素的布局与间距计算。
- **float布局与清除浮动**  
    float常用于传统PC端布局，如双飞翼布局、圣杯布局；清除浮动（clear fix）常通过伪元素或overflow实现。
- **flex画骰子**  
    考察flex容器与项目属性的使用，如justify-content、align-items等，要求能在短时间内完成指定布局。

- **absolute与relative的定位依据**
    - `relative` 是相对于自身原有位置偏移。
    - `absolute` 是相对于最近的定位祖先元素（非static定位）进行定位。

### 响应式相关面试题 
[05:34](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=3264999c3fac4b7f92cd74b6c94752fe#?seek_t=334)

- **REM是什么**  
    REM是相对单位，基于根元素（html）的font-size计算，常用于响应式布局。
- **如何实现响应式**  
    结合媒体查询（media query）与REM动态设置根字体大小，实现不同屏幕尺寸下的自适应显示。
