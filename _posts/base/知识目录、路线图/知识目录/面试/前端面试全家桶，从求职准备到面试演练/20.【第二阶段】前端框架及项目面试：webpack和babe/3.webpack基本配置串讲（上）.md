### 拆分配置与Merge 
[05:42](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=3264999c3fac4b7f92cd74b6c94752fe#?seek_t=342)

- **配置文件结构**
    
    - `webpack.common.js`：公共配置
    - `webpack.dev.js`：开发环境配置
    - `webpack.prod.js`：生产环境配置
- **模式区分**
    
    - `mode: development`：用于开发环境
    - `mode: production`：用于线上环境
- **Merge操作实现**
    
    - 安装`webpack-merge`工具
    - 使用`smart`方法合并公共配置与环境特定配置
    - 示例：
        
        js
        
        ```js
        const { smart } = require('webpack-merge');
        const commonConfig = require('./webpack.common.js');
        
        module.exports = smart(commonConfig, {
          mode: 'development',
          // 其他开发环境相关配置
        });
        ```
        
- **公共配置内容**
    
    - Entry入口路径
    - Module中定义各类loader
    - Plugin中引入HTMLWebpackPlugin
    - 文件路径配置（如SRC目录、输出目录）
- **环境特定配置差异**
    
    - 开发环境设置`window.ENV`为`develop`
    - 生产环境设置`window.ENV`为`production`
    - 开发环境启用devServer，生产环境不启用
    - 图片处理策略不同（如开发环境直接引用URL，生产环境考虑base64编码）

### 启动本地服务 
[10:01](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=3264999c3fac4b7f92cd74b6c94752fe#?seek_t=601)

- **使用webpack-dev-server**
    
    - 安装`webpack-dev-server`
    - 启动命令：`npm run dev` 或 `webpack serve --config webpack.dev.js`
- **关键配置项**
    
    - 端口号设置
    - 自动打开浏览器
    - 是否开启gzip压缩
    - 显示打包进度条
    - 根目录访问路径配置
- **跨域代理设置**
    
    - 场景：前端运行在8080端口，后端接口在3000端口
    - 解决方案：通过proxy代理将API请求转发至后端
    - 示例配置：
        
        js
        
        ```js
        devServer: {
          proxy: {
            '/api': {
              target: 'http://localhost:3000',
              pathRewrite: { '^/api': '' },
              changeOrigin: true
            }
          }
        }
        ```
        
    - 此配置为常见考点，需熟练掌握。
