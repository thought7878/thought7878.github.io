### 前端性能优化知识点清单

---

#### **一、加载性能优化**
1. **减少 HTTP 请求**  
   - 合并文件（CSS/JS 合并）  
   - 使用 CSS Sprites（雪碧图）  
   - 内联小资源（Base64 图片、关键 CSS）  
2. **代码分割与懒加载**  
   - Webpack/Vite 动态 `import()`  
   - 路由级懒加载（React.lazy、Vue 异步组件）  
3. **CDN 加速**  
   - 静态资源托管（JS/CSS/图片）  
   - 多域名分片（突破浏览器并发限制）  
4. **预加载与预请求**  
   - `<link rel="preload">`（关键资源提前加载）  
   - `<link rel="prefetch">`（非关键资源后台加载）  
   - `<link rel="dns-prefetch">`（DNS 预解析）  
5. **首屏优化**  
   - 服务端渲染（SSR）  
   - 静态站点生成（SSG）  
   - 骨架屏（Skeleton Screen）  
   - 关键 CSS/JS 内联  

---

#### **二、渲染性能优化**
6. **减少重排（Reflow）与重绘（Repaint）**  
   - 避免频繁操作 DOM  
   - 使用 `transform`/`opacity` 触发 GPU 加速  
   - 批量 DOM 修改（`documentFragment`、虚拟 DOM）  
7. **防抖（Debounce）与节流（Throttle）**  
   - 滚动、输入、窗口大小调整事件优化  
8. **长列表优化**  
   - 虚拟滚动（Virtual Scrolling）  
   - 分页加载/无限滚动  
9. **动画优化**  
   - 使用 `requestAnimationFrame`  
   - 避免强制同步布局（Layout Thrashing）  
   - CSS 动画优先（硬件加速）  
10. **Web Workers**  
    - 将复杂计算移出主线程  

---

#### **三、资源优化**
11. **图片优化**  
    - 格式选择：WebP/AVIF > JPEG/PNG  
    - 响应式图片（`srcset`、`sizes`）  
    - 压缩工具：ImageOptim、Squoosh  
    - 懒加载（`loading="lazy"`）  
12. **字体优化**  
    - 使用 `font-display: swap` 避免阻塞渲染  
    - 子集化（仅加载必要字符）  
13. **代码压缩**  
    - JS/CSS 压缩（Terser、CSSNano）  
    - 移除未使用代码（Tree Shaking）  
    - Gzip/Brotli 压缩（服务器配置）  
14. **第三方脚本管理**  
    - 异步加载（`async`/`defer`）  
    - 延迟加载非关键脚本（如广告、分析）  

---

#### **四、缓存策略**
15. **强缓存**  
    - `Cache-Control: max-age=31536000`  
    - `Expires`（HTTP/1.1 备用）  
16. **协商缓存**  
    - `ETag`/`Last-Modified`  
17. **Service Worker
