
该函数是 React 内部用于*处理 useState 和 useReducer 钩子状态更新*的核心函数。

函数的主要功能包括：
1. 创建更新对象
2. 根据是否处于渲染阶段决定如何处理更新
3. 实现提前状态计算优化，如果新旧状态相同则跳过渲染
4. 将更新添加到队列并调度更新


- 参数验证：开发环境下检查是否传入了不支持的回调参数
- 获取更新的优先级：获取/请求一个更新优先级
- **创建更新对象**：生成包含优先级、动作（新的状态值、或状态更新函数）等信息的更新对象
- **处理渲染阶段、正常阶段的更新**：区分渲染阶段和正常阶段的更新处理
	- 渲染阶段的更新：将其加入渲染阶段更新队列
	- 正常阶段的更新：
- **状态预计算优化**：尝试预先计算新状态，如无变化则跳过渲染
- **队列管理**：将更新添加到并发队列并调度渲染
- **开发者工具集成**：标记更新供调试使用
