## 前端常用7种设计模式
[07:29](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=f9cedfe7f47d4c2781ece143cbd16a47#?seek_t=449)
    
- **工厂模式** [07:29](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=f9cedfe7f47d4c2781ece143cbd16a47#?seek_t=449)  
	用于解耦对象创建逻辑，统一管理实例化过程。
- **单例模式** [07:31](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=f9cedfe7f47d4c2781ece143cbd16a47#?seek_t=451)  
	确保全局唯一实例（如全局配置管理器、登录态管理器）。
- **观察者模式** [07:33](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=f9cedfe7f47d4c2781ece143cbd16a47#?seek_t=453)  
	实现发布-订阅机制，支撑事件总线、状态管理（如Vue响应式、RxJS）。
- **迭代器模式** [07:33](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=f9cedfe7f47d4c2781ece143cbd16a47#?seek_t=453)  
	封装遍历逻辑，使集合对象与遍历方式解耦；ES6 `Symbol.iterator` 及 `for...of` 语法即原生支持。
- **原型模式** [07:33](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=f9cedfe7f47d4c2781ece143cbd16a47#?seek_t=453)  
	基于原型链实现对象克隆与属性共享，JavaScript 语言原生特性。
- **装饰器模式** [07:33](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=f9cedfe7f47d4c2781ece143cbd16a47#?seek_t=453)  
	动态添加职责，TypeScript 装饰器语法及 React 高阶组件（HOC）为其典型应用。
- **代理模式** [07:33](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=f9cedfe7f47d4c2781ece143cbd16a47#?seek_t=453)  
	为对象提供访问控制层，ES6 `Proxy` API 及 Vue 3 响应式系统底层即基于此。
