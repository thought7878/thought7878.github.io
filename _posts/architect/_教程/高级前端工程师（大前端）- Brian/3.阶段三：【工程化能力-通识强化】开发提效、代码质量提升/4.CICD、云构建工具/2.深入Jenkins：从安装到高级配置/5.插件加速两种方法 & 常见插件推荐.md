### 国内网络环境下插件安装失败的应急处理 
[02:11](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=131)

- **问题现象 [02:11](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=131)**  
    国内安装Jenkins插件常因插件数量多、下载慢、连接超时或校验失败导致安装中断。
- **停止Jenkins服务 [21–22]**  
    执行 `docker compose down` 停止Jenkins容器。
- **清空数据目录 [02:32](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=152)**  
    进入 `data` 目录，执行 `rm -rf *` 清空初始化数据（⚠️高危操作，需谨慎确认）。
- **重启并重新初始化 [24–25]**  
    执行 `docker compose up -d` 重启容器，触发全新初始化流程。
- **获取新初始化密码 [02:56](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=176)**  
    重启后生成新的管理员初始密码。
- **重走初始化流程 [03:06](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=186)**  
    访问 `http://localhost:10050`，输入新密码，点击继续。
- **精简插件安装策略 [28–29]**  
    仍选择右侧“选择插件来安装”，顶部点选“无”，滚动至底部仅勾选Localization: Chinese，点击安装。
- **跳过失败插件 [03:42](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=222)**  
    即使中文包安装失败，也不重试，直接点击“继续”。
- **创建管理员账户 [03:47](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=227)**  
    在分配管理员页面填写信息，点击“保存并完成”。

### 暴力级加速方案：直接修改default.json 
[06:17](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=377)

- **文件定位 [06:17](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=377)**  
    进入Jenkins安装目录 → `updates/` → `default.json`。
- **批量替换URL [57–60]**  
    将文件中所有 `https://updates.jenkins.io/` 或 `https://updates.jenkins-ci.org/` 替换为 `https://mirrors.cloud.tencent.com/jenkins/`。
- **操作替代方案 [61–63]**  
    可通过FTP下载该文件，在VS Code中人工替换，避免记忆复杂命令。
- **替换效果验证 [64–66]**  
    替换后，插件下载速度显著提升，实测SSH相关插件（SSH Credentials、Publish Over SSH、SSH Pipeline Steps）秒级安装。
- **下载性能对比 [67–73]**  
    修改后插件下载呈批量化、高速化：单次下载5个→再下载2个→后续全部“秒下”。

### **必备插件推荐清单** 
[08:10](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=490)

- **GitLab生态插件 [75–81]**
    - GitLab Plugin
    - GitLab API Plugin（若已装GitLab Plugin则无需）
    - GitLab OAuth Plugin（支持GitLab账号直登Jenkins）
- **权限与认证插件 [82–87]**
    - Role-based Authorization Strategy（必选）
    - LDAP Plugin（企业AD/LDAP集成）
    - Matrix Authorization Strategy（矩阵式权限控制）
- **构建与运维插件 [88–92]**
    - Email Extension Plugin（邮件通知）
    - Build Timeout Plugin（构建超时控制）
    - NodeJS Plugin（前端构建依赖）
    - Build User Vars Plugin（构建参数传递）
    - Publish Over SSH（远程部署）
    - Localization: Chinese（已安装）
    - Job Configuration History Plugin（配置历史追踪）

    本课聚焦Jenkins插件系统初始化与加速配置两大主线：

        后者提速效果更显著。
