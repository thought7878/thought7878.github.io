## Registry生效三重保障配置 
[05:17](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=317)
    
- **第一重：全局包安装时指定镜像源** [05:21](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=321)  
	在NodeJS全局配置中，Global NPM packages to install字段修改为：  
	`cnpm --registry https://registry.npmmirror.com pnpm --registry https://registry.npmmirror.com yarn --registry https://registry.npmmirror.com`。
- **第二重：独立NPMRC配置文件** [05:44](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=344)  
	系统配置 → Add a new config → NPM config file → 命名为`NPMRC-conf-customer` → registry值设为`https://registry.npmmirror.com`。
- **第三重：Cypress专用下载镜像环境变量** [06:15](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=1cd97e72b13147e4aca8217600c5deb1#?seek_t=375)  
	系统配置 → 环境变量 → 新增键值对：  
	`CYPRESS_DOWNLOAD_MIRROR = https://npmmirror.com/mirrors/cypress/`。
