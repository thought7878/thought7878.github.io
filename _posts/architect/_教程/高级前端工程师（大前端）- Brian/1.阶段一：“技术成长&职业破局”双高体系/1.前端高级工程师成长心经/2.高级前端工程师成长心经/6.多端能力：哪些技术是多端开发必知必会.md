### 多端跨端开发场景概述 [00:12](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=12)

- **常见多端跨端类型**  
    前端工程师需面对多种开发场景，主要包括以下三类：
    - 官网与管理后台
    - 移动App、小程序与公众号
    - 桌面端应用（Windows、Linux、macOS）

### 第一类：官网与管理后台开发 [00:22](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=22)

- **官网开发要求** [00:24](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=24)
    - 需具备响应式设计能力。
    - 对浏览器兼容性要求高，需适配不同设备与环境。
- **管理后台开发特点** [00:33](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=33)
    - 虽对尖锐性要求不如官网强，但系统复杂度更高。
    - 强调代码组织能力和组件拆分能力。
    - 是检验前端工程师基础能力的重要维度。

### 第二类：移动App、小程序与公众号 [00:53](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=53)

- **移动App** [00:57](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=57)
    - 属于原生应用开发范畴，技术体系独立。
- **小程序与公众号** [01:00](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=60)
    - 具有中国特色的应用形态。
    - 技术实现上属于“webview”模式，*在原生App中嵌入网页*。
    - 使用频率高，技术栈与Web前端相似。
    - 对兼容性要求相对较低。
- **开发优势与意义** [02:12](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=132)
    - 小程序是低成本接入用户的方式。
    - 利用微信平台流量，无需用户下载完整App即可提供服务。
    - 具备部分原生能力，用户体验良好。
    - 显著降低开发成本，提升开发效率。

### 第三类：桌面端应用开发 [01:35](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=95)

- **桌面端应用定义** [01:40](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=100)
    - 运行在 Windows、Linux 或 macOS 上的应用程序。
    - 类似于 QQ、微信客户端，运行于电脑侧而非移动端。
- **主流开发框架** [03:41](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=221)
    - **Electron** [03:44](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=224)
        - 由 Node.js 驱动。
        - 前端使用 HTML、CSS 和 JS 编写界面。
    - **Tauri** [03:47](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=227)
        - 后端使用 Rust 语言编写。
        - API 接口通过 REST 方式进行通信。
        - 前端同学需学习新语言（Rust）方可使用。

### **为何要掌握多端跨端开发能力** [01:58](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=118)

- **市场需求变化** [02:04](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=124)
    - 用户终端从电脑扩展到手机、iPad 等多设备，推动多端适配需求。
- **提升效率与降低成本** [02:12](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=132)
    - 多端复用技术方案可减少重复开发。
    - 小程序等轻量级方案便于快速上线。
- **促进个人成长与职业发展** [02:42](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=7485a33ef186486eb69ef606eb1b703e#?seek_t=162)
    - 技术迁移能力让前端开发者在非传统Web领域继续发挥价值。
    - 掌握新技术带来更广阔的职业发展空间。
    - 技术创新为个人成长提供机会，助力前端之路走得更远。

    - 参与中后台项目时，无法绕开以下两个关键点：
        - 数据库结构设计
        - 系统性能优化
