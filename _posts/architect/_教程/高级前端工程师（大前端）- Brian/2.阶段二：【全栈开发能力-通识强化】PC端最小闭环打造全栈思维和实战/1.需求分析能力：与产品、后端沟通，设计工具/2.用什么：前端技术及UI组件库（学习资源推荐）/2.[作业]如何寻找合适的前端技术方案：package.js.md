## 深入分析package.json获取技术线索 
[01:48](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=108)
    
- `package.json` 是识别项目技术栈的关键文件。
- 其中包含两类核心依赖：
	- **dependencies**：生产环境中使用的正式依赖。
	- **devDependencies**：开发过程中使用的工具类依赖。
- 根据项目类型决定关注重点：
	- 开发lib库工具时，重点关注 **devDependencies**。
	- 使用组件库或功能性库（如admin类库），需同时查看 **devDependencies** 和 **dependencies**。

### 从依赖项中发现主流技术方案 
[03:42](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=222)
    
- 逐项分析 `devDependencies` 中的包及其作用：
	- 以 `types` 开头的包用于提供TypeScript类型定义。
	- 以 `rob plugin` 开头的包为Rob相关插件，例如 `rob plugin commonjs` 可将CommonJS规范转换为ES2015模块。
	- `chokidar`：监视文件变化的库 [04:37](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=277)。
		- 描述为 “Minimal and efficient cross-platform file watching library”。
		- 可用于触发服务重启或编译操作。
	- `cors`：实现跨域资源共享（Cross-Origin Resource Sharing）。
		- 允许前端通过Node.js服务代理访问跨域资源。
	- `dotenv`：读取 `.env` 文件中的环境变量配置。
	- `picocolors`：在终端输出带颜色文本的轻量级工具，用于美化日志输出。

### 拓展认知：CLI工具中常见的美化与交互方案
