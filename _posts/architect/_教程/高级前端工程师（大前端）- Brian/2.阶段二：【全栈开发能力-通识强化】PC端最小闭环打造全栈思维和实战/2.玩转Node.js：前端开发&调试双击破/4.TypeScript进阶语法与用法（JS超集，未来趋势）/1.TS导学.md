### **TS 环境配置 [13:26](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=9798c7c737b94409aa651a40d1a20896#?seek_t=806)**

- **本地环境搭建 [13:31](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=9798c7c737b94409aa651a40d1a20896#?seek_t=811)**
    - 安装命令：`npm install -D typescript nodemon ts-node`
    - 各包作用说明 [14:35](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=9798c7c737b94409aa651a40d1a20896#?seek_t=875)：
        - `typescript`：编译 TS 到 JS 的核心工具。
        - `nodemon`：监听文件变化并自动重启 Node 进程。
        - `ts-node`：直接运行 `.ts` 文件，无需手动编译。
    - 执行方式：安装后使用 `npx ts-node <filename>.ts` 运行 TS 文件。
