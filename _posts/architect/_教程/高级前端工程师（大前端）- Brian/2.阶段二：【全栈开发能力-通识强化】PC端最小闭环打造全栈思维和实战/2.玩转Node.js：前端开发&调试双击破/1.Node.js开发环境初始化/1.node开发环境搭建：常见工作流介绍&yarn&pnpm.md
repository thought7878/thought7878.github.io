## NVM：Node版本管理工具
[00:47](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=47)

- **NVM的概念与通用性 [00:54](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=54)**  
    `NVM`（Node Version Manager）是Node.js的版本控制工具，类似Python和Java中的版本管理工具。
- **NVM的核心功能 [01:17](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=77)**  
    支持在同一台机器上安装并切换多个Node.js版本，例如同时存在Node 16和Node 18，*避免版本冲突*。
- **跨平台差异说明 [01:54](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=114)**  
    macOS/Linux与Windows系统使用的NVM实现不同，需分别处理。

## 完整工作流构建步骤 
[10:32](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=632)

- **标准工作流顺序 [10:35](https://b.quark.cn/apps/5AZ7aRopS/routes/mofb35Rkb?debug=0&fid=845c61eab75a4b2b8c04cea32a174e1f#?seek_t=635)**
    1. 安装NVM并配置Node版本。
    2. 使用淘宝源安装CNPM。
    3. 使用CNPM安装PNPM和Yarn。
    4. 设置NPM默认registry为淘宝源：
        ```shell
        npm config set registry https://registry.npmmirror.com
        ```
        
    5. 此后使用PNPM作为主要包管理工具，其会自动读取NPM配置的registry地址，实现加速下载。
    6. 发布包时临时切换回官方源：
        ```shell
        npm config set registry https://registry.npmjs.org/
        ```
