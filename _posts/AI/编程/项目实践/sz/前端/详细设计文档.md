## 一、文档概述

### 1.1 编写目的

本文档基于《概要设计文档 v1.0》编写，在概要设计模块划分的基础上，*对每个模块进行类/接口级别的详细设计，包括：*
- 核心类与接口定义
- 方法签名、输入输出参数
- 模块内部组件关系（Mermaid 类图）
- 关键交互流程（Mermaid 时序图）
- 状态流转（Mermaid 状态图）

### 1.2 设计原则

- **模块独立性**：各模块通过明确定义的接口交互，内部实现对外透明
- **可测试性**：每个模块可独立进行单元测试，依赖通过注入方式解耦
- **接口契约化**：所有跨模块交互通过 TypeScript 接口定义，确保类型安全

### 2.6 Token 刷新策略

```mermaid
stateDiagram-v2
    [*] --> Active: 登录成功
    Active --> Refreshing: Token 过期前5分钟
    Refreshing --> Active: 刷新成功
    Refreshing --> Expired: 刷新失败
    Active --> Expired: Token 过期
    Expired --> [*]: 跳转登录页

    note right of Active
        Access Token 有效期: 2小时
        自动刷新触发点: 剩余5分钟
        刷新方式: 静息刷新(后台)
    end note

    note right of Refreshing
        并发请求队列: 等待刷新完成
        刷新窗口: Refresh Token 7天内有效
    end note
```

**并发请求处理**：

```typescript
// api/request.ts 中的 Token 刷新拦截器逻辑

let isRefreshing = false;
let refreshSubscribers: Array<(token: string) => void> = [];

function onRefreshed(token: string) {
  refreshSubscribers.forEach(cb => cb(token));
  refreshSubscribers = [];
}

function addRefreshSubscriber(cb: (token: string) => void) {
  refreshSubscribers.push(cb);
}

// 响应拦截器
if (response.status === 401 && !config._retry) {
  if (!isRefreshing) {
    isRefreshing = true;
    try {
      const newToken = await authStore.refreshAccessToken();
      isRefreshing = false;
      onRefreshed(newToken);
      // 重试原请求
      return axiosInstance.request(config);
    } catch {
      isRefreshing = false;
      authStore.logout();
      return Promise.reject(response);
    }
  } else {
    // 将请求加入等待队列
    return new Promise((resolve) => {
      addRefreshSubscriber((token: string) => {
        config.headers.Authorization = `Bearer ${token}`;
        resolve(axiosInstance.request(config));
      });
    });
  }
}
```

## 五、M3 通知与消息模块

### 5.1 模块概述

负责系统通知、数据预警、任务通知的接收、展示和管理。为其他模块提供统一的推送通知能力。

### 5.2 核心接口定义

#### 5.2.1 类型定义

```typescript
// 通知类型
type NotificationType = 'system' | 'alert' | 'task';

// 通知优先级
type NotificationPriority = 'low' | 'medium' | 'high' | 'urgent';

// 通知项
interface NotificationItem {
  id: string;
  type: NotificationType;
  priority: NotificationPriority;
  title: string;            // 通知标题（i18n key 或直接文本）
  content: string;          // 通知内容（支持富文本）
  createdAt: string;        // 创建时间 ISO 8601
  readAt?: string;          // 已读时间
  link?: string;            // 点击跳转链接
  action?: NotificationAction; // 可执行操作
  metadata?: Record<string, unknown>; // 扩展元数据
}

// 通知操作
interface NotificationAction {
  label: string;            // 按钮文本
  type: 'primary' | 'default' | 'danger';
  handler: () => void;      // 点击回调
}

// 通知统计
interface NotificationStats {
  total: number;
  unread: number;
  byType: Record<NotificationType, number>;
  byPriority: Record<NotificationPriority, number>;
}

// 通知过滤选项
interface NotificationFilter {
  type?: NotificationType;
  priority?: NotificationPriority;
  read?: boolean;           // true=仅已读, false=仅未读, undefined=全部
}

// 通知分页查询参数
interface NotificationQueryParams {
  filter?: NotificationFilter;
  page: number;
  pageSize: number;
}
```
