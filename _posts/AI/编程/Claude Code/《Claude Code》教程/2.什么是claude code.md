## Claude Code核心定位
是*为AI大模型配备轻量级工具框架与运行环境，辅助完成复杂编程任务*的主动型AI编程工具。

## 主动型系统构成
包含**AI模型、工具集、运行环境**，模型借助工具遍历代码库、处理多任务。

## 核心能力方向
核心是代码库的发现、解释与设计，还支持代码编写、重构、调试，适配编程与数据分析场景。

## 内置基础工具
提供文件读写、编辑、模式查找、联网搜索、创建子代理、执行shell命令等工具。

## 扩展能力
支持*通过MCP（模型上下文协议）连接服务器，开源、模型无关的协议*，可**方便数据和人工智能系统之间的交流，拓展多样功能**。


## 安全与搜索特性
采用主动型搜索，无需上传代码库、不索引全部代码，数据留在本地开发环境，降低安全风险。

## 记忆功能机制
**通过cloud.md文件保存配置与对话记录，本地存储**，可随时清除或恢复对话。
