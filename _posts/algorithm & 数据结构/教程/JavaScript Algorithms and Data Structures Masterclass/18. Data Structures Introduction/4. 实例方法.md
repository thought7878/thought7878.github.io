Data Structures: Adding Instance Methods


视频围绕JavaScript中数据结构的实例方法展开讲解，通过学生类和单向链表类的案例，详细*介绍了实例方法的定义、作用及实际应用*，具体如下：

- **实例方法概念与常见示例**：[00:00]说明`实例方法`是课程中频繁使用的方法类型，*作用于特定实例并提供相关功能*。*以数组为例*，创建数组实例data后，调用data.push()等方法，*这些方法均作用于data这个数组实例*，同时提及后续会在学生类和单向链表类中定义实例方法。

- **学生类中简单实例方法定义**：[01:26]在学生类中定义fullName实例方法，该方法返回包含学生firstName和lastName的字符串，其中this指代当前学生实例，不同学生实例调用该方法会因名字不同得到不同输出。
- **学生类中更新属性的实例方法**：[03:24]为学生类添加tardies（迟到次数）属性，默认值为0；定义markLate（标记迟到）实例方法，实现tardies加1的功能，初始返回包含学生姓名和迟到次数的字符串，后续优化为当tardies≥3时返回“你被开除了”。
- **学生类中处理分数的实例方法**：[06:07]为学生类添加scores（分数）属性，默认值为空数组；定义addScore实例方法，接收新分数并添加到scores数组末尾，同时返回该数组，强调通过方法操作数据是更常规的做法；还定义calculateAverage实例方法，利用reduce方法计算scores数组中分数的总和，再除以数组长度得到平均值。
- **提及后续类方法讲解**：[09:38]在介绍完实例方法后，说明接下来会简要讲解类方法，指出类方法使用频率不高，但作为ES2015中类的相关内容仍需涵盖。
