## HTML 和 DOM 中的树结构应用

[00:00]
HTML 天然适合树状结构，**元素间存在父子关系，且层层嵌套**。
浏览器接收请求*解析 HTML 后会生成文档对象模型（DOM）*，在 Chrome 调试工具中可交互查看不同节点，且*每个节点对应一个 JavaScript 对象*，通过相关代码可访问如 body 等节点及其属性、方法和子元素。

## 网络路由中的树结构应用

[01:48]
网络路由后台使用树结构，维基百科路由页面有各类树状图示，用于*展示不同传输方案及消息如何传输到节点或多个节点*，但视频作者未深入解释其具体工作原理。

## 抽象语法树的应用

[02:23]
`抽象语法树`是**用树状结构描述编程语言语法**的方法，无论何种语言都**可通过其辅助处理代码**，如*运行代码、解析代码、确保语法有效*，视频以 while 循环为例简单说明其结构，未深入细节。

## 人工智能和机器学习中的树结构应用

[03:16]
**树结构在该领域应用广泛**，最简单的例子是极小化树。以井字棋游戏 AI 开发为例，可将游戏逻辑分解为树状结构，描述棋盘当前状态下人类玩家可能的移动及计算机对应的最佳移动，这种树结构也被称为`决策树`，在复杂分类算法中也有应用。

## 操作系统文件夹的树状建模

[04:47]
mac 或 pc 等操作系统中的**文件夹以树状结构建模**，呈层级化，有顶层文件夹（如 mac 的根目录），路径明确，文件夹内可包含其他文件夹和文件，结构中存在分支，子节点又有自己的子节点。

## JSON 中的树结构

[05:46]
JavaScript 对象表示法（JSON）在使用过程中，*当从 API 获取数据并将响应从字符串解析为对应语言（如 JavaScript、Python）的对象时，会涉及遍历树或创建树状结构*，JSON 数据具有父节点和子节点，且最终只有一个根节点，非常适合用树来表示。
