## 二叉搜索树核心特性介绍

[00:00]
说明**二叉搜索树组织节点的规则**，*即右边所有节点大于父节点，左边所有节点小于父节点*，**该特性让`查找`容易且快速，也便于`插入`新元素找到合适位置，相比无序树搜索更高效**。

## 二叉搜索树搜索过程演示

[00:31]
以*查找数字 72* 为例，演示从根节点开始的搜索步骤。
先比较 72 与根节点 41，因 72 大于 41 *关注右半部分*；
再比较 72 与 65，72 大于 65 *继续转向右侧*；
最后比较 72 与 91，72 小于 91，*完成查找*。
同时指出**每次比较大致将需检查的值数量减半**，*类似列表二分查找原理*（数据有序排列），但*实际拆分情况取决于树的分布，总体平均比较次数或节点数量减半*。

## 二叉搜索树与无序树查找对比

[01:38]
*若为无序树*，查找 72，*其可能在树的任何位置，需检查每个节点遍历整棵树*，遍历方式多样；
*而二叉搜索树查找有具体步骤，通过比较转向左右、不断减半范围*，直至找到目标或确定目标不存在。

## 二叉树与二叉搜索树概念总结

[02:21]
`二叉树`中*每个节点最多有两个子节点*，存在多种类型，`二叉搜索树`是其中之一，*适用于排序数据或可排序数据*（存在顺序关系、可比较），*除遵循二叉树“每个父节点最多有两个子节点”的规则外，还具有“父节点左侧所有节点小于父节点，右侧所有节点大于父节点”的特性*，**该特性保障了高效搜索，能快速找到目标**。
