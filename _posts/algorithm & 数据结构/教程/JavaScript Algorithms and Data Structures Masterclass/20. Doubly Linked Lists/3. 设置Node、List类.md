Setting Up Our Node Class


视频讲述了在JavaScript中*搭建双向链表所需的节点类、双向链表类*的基础框架，以及简单测试节点间双向链接的方法，具体如下：

## 双向链表与单向链表的异同
[00:00]
**两者基础结构相似**，节点都包含值，双向链表类也包含*头节点、尾节点、长度*；
**不同之处**在于双向链表的*节点新增了previous属性*，后续添加的*逻辑代码与单向链表差异较大*，这也是专门讲解该部分的原因。


## 定义Node类构造函数
[00:43]
节点类构造函数中，设置this.val存储节点值，this.next初始化为null（指向后一个节点），this.previous初始化为null（指向前一个节点），这是双向链表节点与单向链表节点唯一的不同点。


## 定义双向链表类构造函数
[01:05]
双向链表类构造函数保持为空，仅设置this.head（`头节点`）为null，this.tail（`尾节点`）为null，this.length（`链表长度`）初始化为0，完成双向链表的骨架代码搭建，且后续不再修改节点类，所有内容添加到双向链表类。


## 简单测试节点间双向链接
[01:50]
创建值为12的first节点，将first.next指向值为13的新节点，再设置first.next.previous指向first，实现两个节点间的双向链接，可在这两个节点间来回切换，验证双向链表节点链接的有效性。
