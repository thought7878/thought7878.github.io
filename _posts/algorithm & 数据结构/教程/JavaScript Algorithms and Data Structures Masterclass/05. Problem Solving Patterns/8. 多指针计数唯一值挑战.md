8. Multiple Pointers Count Unique Values Challenge


视频讲述了如何使用双指针模式解决“统计有序数组中唯一值数量”的问题，具体如下：

## 问题介绍与基本要求
[00:00]
需实现名为countUniqueValues的函数，该函数接收一个已排序数组（可能含负数），返回数组中唯一值的数量，并举例说明不同场景下的预期结果，如空数组无唯一值，含-2、-1、0、1的数组有4个唯一值等。

实现一个名为countUniqueValues的函数，该函数接收一个排序后的数组，并统计数组中的唯一值数量。数组中可能存在负数，但数组始终是排序好的。

## 双指针模式思路提示
[01:18]
提示可使用双指针解决该问题，指出指针并非从两端向中间推进，而是根据条件朝特定方向移动，还提及可尝试自行解决，若需要建议，可参考在数组开头构建唯一值的方案，无需单独变量统计。


## 双指针具体操作演示
[01:59]
以具体数组为例，设置指针i和j，j作为“侦察兵”先行移动。当i和j指向的值相同时，仅移动j；当值不同时，先移动i，再将j指向的值放入i当前位置，在数组开头构建唯一值，直至j到达数组末尾。


## 时间复杂度与结果计算
[03:39]
说明该方法仅需一次遍历数组，时间复杂度为O(n)。最终唯一值的数量可通过指针i的索引计算，即i的索引加1（因索引从0开始），同时提到若不能修改数组，此方法并非最佳选择。


## 其他可行方向提示
[04:17]
指出除从左侧开始使用双指针外，也可尝试从右侧开始并向左移动双指针，后续会在练习结束后提供完整解决方案。
