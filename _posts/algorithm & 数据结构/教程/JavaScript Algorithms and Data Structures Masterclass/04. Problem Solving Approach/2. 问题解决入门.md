## 算法定义
[01:21]
`算法`在当下应用广泛，**本质是*完成任务的步骤集合***，既可以是*计算前一百个质数这类简单的数学步骤*，也可以是*像Facebook广告推荐、谷歌搜索结果排序*这类*由众多开发者长期协作开发的复杂系统相关步骤*。

## 学习算法的意义
[02:42]
*编程中的几乎所有操作都涉及算法*，无论是基础事务还是构建复杂应用，都需运用算法相关知识；
且*所学内容适用于多数编程领域*，尤其对*面试场景*帮助显著，能让人在面试中*冷静应对陌生难题*。

![[_posts/algorithm & 数据结构/教程/JavaScript Algorithms and Data Structures Masterclass/04. Problem Solving Approach/media/6ea8f198437dfe5eca0b07c946422b95_MD5.jpeg]]

## 提升问题解决能力的途径
[03:50]
`问题解决能力`并非完全与生俱来，*通过练习可提升*，主要有*两个途径：*
- 一是**制定问题解决计划**，涉及*面对问题、分解问题*的策略；
- 二是**掌握常见问题解决模式**，很多面试中的*算法问题可归类*，识别类别后能*借助可复用模板解题*。

![[_posts/algorithm & 数据结构/教程/JavaScript Algorithms and Data Structures Masterclass/04. Problem Solving Approach/media/24463587e0fc8405fcf4bd661ffef585_MD5.jpeg]]

## 问题解决的具体步骤
[06:20]
**解决问题需遵循一系列步骤**，包括**理解问题、探索具体示例、分解问题、解决/简化问题、回顾并重构**，*每个步骤都有独立视频讲解*，共同构成解题策略和挑战方案。
