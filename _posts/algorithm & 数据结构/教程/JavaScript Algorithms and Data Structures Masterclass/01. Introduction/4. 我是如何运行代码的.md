4. How I'm Running My Code

视频讲述了*课程中使用的JavaScript代码运行方式*，重点介绍了Chrome浏览器的Snippets工具及其他可选工具，同时说明了课程对学习者的基础要求，具体如下：

- **课程开发环境选择原则**：[00:00]为课程尽量保持开发环境简单，无需下载任何软件，所有操作在浏览器中完成；但这并非强制要求，学习者可继续使用自己熟悉的工具。
- **课程基础要求**：[00:15]唯一要求是学习者了解基本的JavaScript语法，且已熟悉编写JS代码，具备该基础才能更好地学习课程。
- **Chrome浏览器Snippets工具介绍**：[00:29]Chrome自带JavaScript代码片段工具Snippets，打开方式可通过Mac快捷键Command+Option+J，或进入检查-元素面板-切换到源代码标签，在其下找到代码片段子标签；该工具在课程中会被多次使用。
- **其他JS代码运行工具推荐**：[01:21]除Chrome Snippets外，还可使用Sublime Text编写代码、其他具备类似功能的浏览器，以及JS Bin、CodePen等平台，有多种选择可用于运行JavaScript代码。
- **Chrome Snippets新建与命名**：[01:34]在Snippets中可新建空代码片段，课程中会多次演示该操作，示例中暂时将新建片段命名为“演示片段”，新建后可调整片段界面大小。
- **Chrome Snippets代码编写与执行**：[01:49]可在片段中编写简单代码（如console.log("来自片段的问候")）；执行代码可点击播放按钮，也可使用快捷键（Mac上为command+回车，PC上会提示对应快捷方式），按下快捷键后能看到代码输出结果。
- **Chrome Snippets在课程中的实际应用示例**：[02:22]课程中会多次使用该工具运行代码，例如运行包含二叉搜索树广度优先搜索相关的代码片段，以及为不同功能（如双向链表添加set方法、remove方法）分别新建片段。
- **Chrome Snippets工具优势**：[02:57]这是一种在浏览器中快速运行代码的简便方法，无需下载任何东西，且不区分Mac或PC设备。
