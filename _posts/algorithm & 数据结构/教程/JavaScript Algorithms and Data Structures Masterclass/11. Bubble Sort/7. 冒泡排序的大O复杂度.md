7. Bubble Sort BIG O Complexity


视频讲述了冒泡排序的时间复杂度相关知识，包括一般情况、最佳情况、优化方式及适用场景等内容，具体如下：

- **冒泡排序一般时间复杂度**：冒泡排序一般时间复杂度是n的平方（O(n²)），原因是其采用嵌套循环，大致会对数组中的每个元素进行一次比较，例如为将元素8移动到数组末尾，需将8与数组中每个其他元素比较。
- **冒泡排序最佳时间复杂度**：当数据几乎有序或已经有序时，在添加了无交换变量以提前终止的优化版本冒泡排序中，时间复杂度接近线性，最佳情况为O(n)。以处理8个元素为例，虽需两次遍历（第一次遍历，第二次遍历确认无交换后退出循环），但整体可简化为O(n)。
- **冒泡排序适用场景与后续内容提示**：除非明确知道数据几乎有序，否则不建议使用冒泡排序；视频还提及后续会介绍另一个可作为候选的排序算法，同时提到该视频中涉及不使用复杂交换语法的冒泡排序版本、使用ES2015语法的版本以及添加无交换变量优化的版本。
