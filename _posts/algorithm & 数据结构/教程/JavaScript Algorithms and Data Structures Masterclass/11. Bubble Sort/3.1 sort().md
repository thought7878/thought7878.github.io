JavaScript 中的 `Array.prototype.sort()` 方法用于对数组的元素进行排序，并返回已排序的数组。默认情况下，`sort()` 方法会将元素转换为字符串，并按照字符串的 UTF-16 码元值进行升序排序。

### 基本语法

```javascript
arr.sort([compareFunction])
```

- `compareFunction`（可选）：一个用于确定数组中元素顺序的函数。如果省略，则将元素转换为字符串，并按字典顺序排序。

### 比较函数

比较函数 `(a, b)` 的返回值决定了排序顺序：

- 如果 `compareFunction(a, b)` 小于 0，则 `a` 排在 `b` 之前。
- 如果 `compareFunction(a, b)` 大于 0，则 `b` 排在 `a` 之前。
- 如果 `compareFunction(a, b)` 等于 0，则 `a` 和 `b` 的相对位置不变。

#### 示例

1. **数字排序（升序）**：

```javascript
const numbers = [3, 1, 4, 1, 5, 9, 2, 6];
numbers.sort((a, b) => a - b);
console.log(numbers); // [1, 1, 2, 3, 4, 5, 6, 9]
```

2. **数字排序（降序）**：

```javascript
const numbers = [3, 1, 4, 1, 5, 9, 2, 6];
numbers.sort((a, b) => b - a);
console.log(numbers); // [9, 6, 5, 4, 3, 2, 1, 1]
```

3. **字符串排序（默认）**：

```javascript
const fruits = ['banana', 'apple', 'cherry'];
fruits.sort();
console.log(fruits); // ['apple', 'banana', 'cherry']
```

4. **字符串长度排序（升序）**：

```javascript
const words = ['banana', 'apple', 'cherry', 'kiwi'];
words.sort((a, b) => a.length - b.length);
console.log(words); // ['kiwi', 'apple', 'banana', 'cherry']
```

5. **对象数组排序**：

```javascript
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Alice', age: 35 }
];

// 按年龄排序（升序）
people.sort((a, b) => a.age - b.age);
console.log(people);
// [
//   { name: 'Jane', age: 25 },
//   { name: 'John', age: 30 },
//   { name: 'Alice', age: 35 }
// ]
```

### 注意事项

- `sort()` 方法会**直接修改原数组**，而不是创建一个新的排序后的数组。
- 如果不提供 `compareFunction`，元素会被转换为字符串再进行排序，这可能导致不符合预期的结果（例如，数字 `10` 会排在 `2` 之前，因为字符串 `'10'` 在字典顺序上小于 `'2'`）。
