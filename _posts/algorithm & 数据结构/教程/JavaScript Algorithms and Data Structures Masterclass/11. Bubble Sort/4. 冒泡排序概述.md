## 核心思想、名称由来
[00:34]
`核心思想`是**对需升序排列的数组，让较大数值逐个“冒泡”到顶部**（也被称为*下沉排序*，即最大值沉到末尾）。
同时提及可借助“https://visualgo.net”这一浏览器内的交互工具逐步演示排序算法。

一种排序算法，其中最大值会“冒泡”到顶端！

## 工作原理演示
[01:39]
通过循环遍历数组元素，将当前元素与下一个元素比较，若当前元素更大则交换。以含七个元素（数值10到37）的数组为例，展示第一次遍历过程，最终最大值升至顶端，且后续会重复该过程，随着排序进行，需排序的元素数量逐渐减少。


## JavaScript中数组元素交换的两种方法
[04:38]
一是**传统方法**，设置临时变量存储其中一个值，再依次更新两个索引对应的数组内容，该方法易读易懂；
二是**ES2015语法**，可在一行内实现交换，*适合追求文件体积的情况*，同时建议定义swap函数使冒泡排序代码更简洁。

许多排序算法都涉及某种类型的交换功能（例如，交换两个数字以使其按顺序排列）

## 冒泡排序代码实现思路
[06:12]
定义bubbleSort函数，接收全为数字的数组，外层循环从数组末尾开始（变量i），用于逐步缩小排序范围，内层循环变量j从开始到i减一，在循环中比较数组j项与j+1项，若数组j项更大则调用交换方法，最后返回排序好的数组，后续视频将展示具体实现。

**我的伪代码：**
- 需要两个循环，
- 内部循环：要比较相邻两元素大小
	- 如果左侧元素>右侧，交换两元素的位置，当前指针元素右移一位（j++就可以）
	- 否则（左侧元素<=右侧），当前指针元素右移一位（j++就可以）
	- 循环终结条件：最多次数是<=arr.length-1 或 <arr.length，最佳值是<arr.length-i外部循环次数（外部循环完成一次就找到了未排序最大值，下次内部循环没必要再跟上次值比较了，上次已经比较了）
- 外部循环：
	- 每次从数组头开始，进行内部循环
	- 循环终结条件：最多次数是<=arr.length-1 或 <arr.length，

```js
function bubbleSort(arr) {

  for (let i = 0; i < arr.length; i++){
    
    for (let j = 0; j < arr.length - i; j++){
      if (arr[j] > arr[j + 1]) {
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
      }
    }
  }

  return arr;
}

console.log('排序结果：',bubbleSort([3,4,1,5,2]));
```

**冒泡排序伪代码：**
- 从数组的末尾开始，用一个名为i的变量向前循环
- 用一个名为j的变量从数组的开头开始进行内部循环，直到i-1的位置
- 如果arr[j]大于arr[j+1]，就交换这两个值！
- 返回排序后的数组
