### 解释默认排序依据
[01:01]
引用MDN文档说明，*数组默认排序依据*是字符串的Unicode码点，*排序时每个数组元素会先转换为字符串，再取其Unicode值进行排序*。
*这种方式几乎不是用户想要的，除非处理字符串且特定场景*。


## sort方法的比较器函数及作用
[01:45]
sort方法接受可选的比较器函数，该函数接收a、b两个元素，根据返回值确定排序顺序。返回负数时，a在b之前；返回正数时，a在b之后；返回零时，a和b视为相同。

参考：[[3.1 sort()]]

**告诉JavaScript如何排序：**
- 内置的sort方法接受一个*可选的比较函数*
- 你可以使用这个比较函数来*告诉JavaScript你希望它如何排序*
- 比较函数会查看成对的元素（a和b），*根据返回值确定它们的排序顺序*
	- 如果返回一个负数，a应该排在b前面
	- 如果返回一个正数，a应该排在b后面
	- 如果返回0，就排序而言，a和b是相同的
