## 明确reverse方法实现要求
[00:24]
要求*对链表进行就地反转，不需要创建副本，无需复制并返回新链表，直接在原地遍历并反转节点*。

## 展示单向链表反转具体操作示例
[00:44]
以包含头节点（thirteen）、two seven、three two、seventy one（尾节点）的单向链表为例，说明反转过程。从头部开始，先将头节点设为尾节点，处理two seven时，先暂存其原next节点three two，再将two seven的next指向thirteen；处理three two时，暂存其原next节点seventy one，再将three two的next指向two seven；处理seventy one时，将其next指向three two，最终seventy one成为新头部。

## 讲解reverse方法伪代码思路
[02:24]
方法无参数，首先交换链表的头和尾，
然后创建next、previous、node（或current，初始化为头部）三个变量。
遍历列表时，先将next设为当前node的next属性以保存原连接，再将node的next设为previous，之后更新previous为当前node，node设为next，以此实现节点反转。


我的伪代码（不准确）：
- 函数reverse，输入是list，输出是reversedList
- 边界条件：list为null/undefined/length=0，return null/undefined；list为只有1个元素，
- 循环遍历list，while的终止条件是current != null
	- 如果current是head，current.next=null
	- 如果current是tail，
	- 其他情况，
		- 保存前一个节点（previous），保存下一个节点（next）；
		- 将当前节点的下一节点指向前一节点，current.next=previous；
		- 更新previous为current，更新current为next，更新next为next.next；

- 函数reverse，输入是list，输出是reversedList
- 边界条件：list为null/undefined/length=0，return null/undefined；list为只有1个元素，
- 循环遍历list，while的终止条件是current != null
	- 如果current是head，current.next=null
	- 如果current是tail，
	- 其他情况，
		- 保存前一个节点（previous），保存下一个节点（next）；
		- 将当前节点的下一节点指向前一节点previous，current.next=previous；
		- 更新previous为current，更新current为next，更新next为next.next；

p  c  n 
   p  c  n
      p  c  n 
         p  c    n
*/

reverse() {
    let list = this;
    
    if (!list) {
      return undefined;
    } else if(list.length===1) {
      return list;
    }  

    let prev = null;
    let current = list.head;
    let next = list.head.next;
  
    while (current) {
      current.next = prev;
      prev = current;
      current = next;
      next = next?.next;
    }
    
    let temp = this.head;
    this.head = this.tail;
    this.tail = temp;

    return list;
  }
}
