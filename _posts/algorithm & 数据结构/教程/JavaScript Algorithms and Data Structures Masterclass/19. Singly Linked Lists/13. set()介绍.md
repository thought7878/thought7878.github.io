Singly Linked List: Set Intro


视频讲述了单向链表（Singly Linked List）中*set方法*的相关介绍，包括*方法参数、功能、实现思路*及后续安排，具体如下：

## set方法参数、功能说明
[00:00]
set方法接受*两个参数*，分别是`位置（索引）`和要更新到该索引处`元素的值`，**作用**是*更新链表指定索引位置元素的值*，*例如*，可将索引1处节点的值更新为“hello world”。

## 函数定义规范
[00:39]
需定义一个接受两个参数的函数，虽参数顺序可调整，但要保持一致性，视频中更倾向于*将索引参数放在前面，值参数放在后面*。

## set方法实现依赖说明
[00:51]
实现set方法可依赖上一视频定义的get方法，*通过get方法找到指定索引对应的节点*，若节点不存在，get方法会处理该情况（如更新索引为##10的不存在节点）；若找到节点，则更新其值。

## set方法返回值规则
[01:11]
若未找到指定节点，set方法*返回false*；
若成功找到节点并更新其值，方法*返回true*。
