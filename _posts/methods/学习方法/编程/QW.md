### 2. **你目前的理解程度（可选）**
> 示例：初学者 / 有一定基础 / 准备面试 / 实际项目中遇到问题

```

```

---

### 3. **具体问题描述**
> 尽量清晰描述你要解决的问题或想要实现的功能

```

```

---

### 4. **是否有相关代码或错误信息？（可选但推荐）**


### 5. **你已经尝试过的解决方法（可选）**
> 帮助我更快定位问题所在


### 6. **你希望获得的帮助类型**

