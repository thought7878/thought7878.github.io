## Sass 部分文件特性及使用

[01:28]
Sass 支持`模块化（Partials）`文件，可*创建含 CSS 片段的源文件*供后续引入其他源文件，**利于模块化和重用 CSS，方便维护**；
*模块化（Partials）文件名必须以下划线开头*，表明其为部分文件且不应编译为 CSS；
导入部分文件无需指定下划线，系统会自动判断；
若存在同名且一个以下划线开头、一个没有的文件，Sass 编译时会报错。

## 添加 Sass 处理规则

[04:01]
需*注意加载器在数组中的顺序，Webpack 从右到左处理加载器*，**先调用 sass-loader将 sass 转换为 css，再调用 css-loader将 css 转换为 JavaScript 表示，最后调用 style-loader在 HTML 页面创建样式标签并放入 css**。
